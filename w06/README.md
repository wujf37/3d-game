介绍

完成了牧师与魔鬼2.0动作分离版，点击牧师、魔鬼、船会有相应的动作，前提是动作符合条件。在游戏过程中若游戏结束（失败或者胜利）则游戏结果会显现并且会出现restart按钮，点击restart重启游戏。由于是动作分离版，游戏对象的移动由专门的动作控制器实现。



#### 工程的部署方式

将代码中的Assets文件夹替换掉新项目中的Assets文件夹，运行。
